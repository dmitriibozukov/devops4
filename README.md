# Отчёт: Логирование и мониторинг
## Выполненные задачи

### 1. Развёртывание системы мониторинга
- Установлен Docker Compose
- Настроены сервисы: Spring Boot, Prometheus, Grafana, Loki, Node Exporter

### 2. Конфигурация
- Создан `docker-compose.yml` для оркестрации контейнеров
- Настроен сбор метрик приложения через Actuator
- Реализовано логирование через Loki

### 3. Визуализация
- Создан дашборд Grafana (`grafana.json`)
- Настроены графики использования памяти JVM и хоста
